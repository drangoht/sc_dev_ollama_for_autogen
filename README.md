>> (venv) ) vscode ➜ /workspaces/sc_dev_ollama_for_autogen (master) $

>> python --version

>> cat /etc/os-release

    Debian GNU/Linux 11 (bullseye)

>> ollama -v

    ollama version is 0.3.13

>> ollama list

    llama3.2:latest
